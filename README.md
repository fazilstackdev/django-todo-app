# Todo Django Project

## Setup
python -m venv env

pip install -r requirements.txt
python manage.py migrate
python manage.py runserver

# 📝 Django ToDo Application

A simple and clean ToDo web application built with Django.

## 📌 Features
- Create tasks
- Update tasks
- Mark tasks as completed
- Delete tasks
- Clean UI using Django Templates

## 🛠 Technologies Used
- Python 3
- Django 5.x
- SQLite (default Django DB)
- HTML / CSS

git clone https://github.com/USERNAME/todo-django.git
